# Part 1: Python Basics & Control Flow

## Overview
This project focuses on the fundamentals of Python, including variable management, string manipulation, and control flow (if/else statements and loops).

## Features

**Grade Cleaning:** Processes raw, messy string data into usable integers.
**Performance Tracking:** Calculates average scores and identifies students who need support.
**Formatted Output:** Prints clean "Student Profile Cards" for easy reading.


## File List

`part1_grade_tracker.py`: The main script containing the logic.
