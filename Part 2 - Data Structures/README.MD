# Part 2: Data Structures

## Overview
This task demonstrates the use of complex Python data structures like nested dictionaries and lists to manage a real-world scenario: a restaurant ordering system.

## Features

**Menu Management:** Stores categories, prices, and availability.
**Inventory Tracking:** Uses `deepcopy` to create a safe backup of stock levels before processing orders.
**Sales Analysis:** Aggregates daily revenue and identifies the best-selling items.


## File List

`part2_order_system.py`: Script managing the menu, cart, and sales log.
